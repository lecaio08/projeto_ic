# projeto IC: PCVGS.boy

## descrição do projeto
- o projeto consiste em simular um fliperama, contemplando os classicos jogos car race, snake e pong.

## materiais e peças Necessárias
- arduino mega 2560
- display lcd 16x2
- 5 botoes
- carcaça em acrílico
