# projeto IC

## descrição do projeto
- o projeto consiste em simular um videogame simples, contemplando os classicos jogos snake, pong e car race

## materiais e peças Necessárias
- arduino nano
- display lcd 2x16
- 5 botoes
- carcaça feita com cortes em acrílico
